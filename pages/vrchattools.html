FORMAT GUIDE

TEXTMESHPRO

Uses TextMeshPro rich text tags like <b>, <i>, <color>, and <link>. Perfect for in-world text displays with clickable links.

PLAIN TEXT

Simple, unformatted text. Great for basic world descriptions or documentation where formatting isn't supported.

MARKDOWN

Uses Markdown syntax (**, *, [links]). Ideal for GitHub repos, Discord messages, or documentation sites.

BBCODE

Uses BBCode tags like [b], [i], [url]. Common in forums and some social platforms.

BEST PRACTICES

RESPECT LICENSES

Always check and follow asset licenses. Include license info when required by the creator.

LINK TO SOURCES

Provide links to asset creators' stores, portfolios, or social media to support their work.

BE SPECIFIC

Name specific assets, not just "various models". This helps others find resources and shows appreciation.

KEEP IT UPDATED

Update credits when you add, remove, or replace assets in your world. Accuracy matters!
